This is our info fetch app.

This uses IntelliJ with Java JDK8u172.

Maven Project with Java using Spring Boot v2.0.2

H2 Database is used to store random information. PyAppFetch is used to generate a sqlOutput file which contains
random information. The app will initialize its H2 database from the sqlOuput file.

This allow us to enter an ID number and the app will fetch details on that ID number.
	Details:	Person Name & Occupation
